# P 블록 — 성능 개선 실행 결과 (2026-07-10; 설계 = NEXT_STEPS_DIAGNOSIS.md)

> **정정 (2026-07-13, codex 외부 감사 수용 — fable 재계산으로 전 항목 검증,
> `scripts/v_codex.py`):**
> ① **2022-25 헤드라인(0.689/0.693) 철회.** A1의 2025 포함은 자체 감사
> E0A(라벨 신뢰 종료 2024-12-31, 늦은 등재 1.5-2년 소급, "2025-26 무조건
> 제외")를 위반한 오류 — 시트 최신 날짜와 수집 완결성을 혼동했다. 정정
> 규칙 `t+H ≤ 2024-12-31` 적용 시 **대표 수치 = H90 ROC 0.701 [0.643,
> 0.759] (사건 75) / H150 0.696 [0.645, 0.746] (사건 80, 2024-09 창 제외)**.
> 이 수치는 "adaptive selection 이후 **조건부 backtest**"로 표기한다(전
> 실험이 같은 test를 반복 조회, untouched test 부재). **2025 창은 라벨
> 성숙(~2027 중반 스냅샷) 후 동결 모델의 진짜 전향 확인 세트로 재분류.**
> ② 아래 hazard 절의 "수술 이중계상 제거"는 **과장 — 철회.** landmark 내
> 검열만 하며 겹치는 landmark 간에는 같은 수술이 반복된다(양성 구간 252 vs
> distinct 79, 10계수 실질 EPV ~7.9). cluster-aware/사건 가중 민감도가 필요.
> ③ "확률 보정 확보"도 **과장 — 철회.** 실측: 평균 예측 0.98%/1.46% vs
> 유병률 1.40%/1.97%, top-decile ~2.2× 과소예측, 재보정 slope 1.86/1.90 —
> class-weight 인플레 제거일 뿐 보정이 아니며 **validation 재보정 필수**.
> ④ "무작위 대비 2-3배 포착"도 **과장 — 철회.** 사건이 양성 월을 2-3개
> 가져 무작위도 복수 기회를 얻음. MC 2000회 실측 lift = **1.76× (H90) /
> 1.55× (H150)**, 단 둘 다 무작위 95% 상한 밖(P≤0.012)으로 유의.
> ⑤ "불펜 구조적 불가"(진단 문서)도 **완화 — 철회.** 성숙 test에서 RP-내
> ROC 0.65/0.65(무작위 아님), 문제는 top-50이 사실상 전부 SP에 배정되는
> **경보 배분**(RP 사건 포착 1/37, 1/40). 역할별 경보 quota는 Pareto 선택
> 문제로 재규정. ⑥ seen/novel 투수 ROC 0.706/0.683 (H90), 0.707/0.661
> (H150) — **KBO 이전 기대치는 novel 쪽(~0.66)으로** 설정.
> 회고 F1@0.5는 임의 작동점(threshold 미구현)이므로 배치 성능으로 인용
> 금지. 이하 본문은 이력 보존을 위해 원문 유지.

순서: P0-1(2024 확장) → P0-2(구조 교정) → P1(미니 티어) → P2(모델 형태).
전 단계 사전 등록 paired 검정, 동결 프로토콜(clustered bootstrap seed 0 공유,
event recall@k), 각 단계 게이트 통과. 코드 `scripts/p0_*.py, p1_tiers.py,
p2_hazard.py`, 수치 `p0_*.csv, p1_tiers.csv, p2_hazard.csv`.

## 최종 canonical (v4): discrete-time hazard supermodel

- **데이터**: cohort_v3 (결정일 2017-2024 Apr-Sep, test 2022-24).
  `data/prospective/{slim_games_v3, game_features_v3, cohort_v3}.parquet`.
  v2 대비 게이트: ≤2023 부분집합 byte-동일 (G1a/G1b/G2), fold 양성 동일 (G3).
- **feature (M_sa, 9개)**: M-role 6 + prior_pc_rate(직전 시즌 총투구/183) +
  ncg_log(경력 경기수 log) + vt_missing(구속 추세 결측 지표).
- **형태**: 30일 person-period 확장(s=0..4, at-risk 검열) + plain LR
  (M_sa+s, 10계수) → P(H) = 1−∏(1−h_s). 라벨 정합 검증(3빈 누적 ==
  label_H90_B0, 5빈 == label_H150_B0).
- **수치 (test 2022-24, 사건 75/80)**:
  | | H=90 | H=150 |
  |---|---|---|
  | ROC | **0.701** | **0.708** |
  | M_sa 이진 (비교 anchor) | 0.692 [0.640,0.746] | 0.704 [0.650,0.758] |
  | M-role 구(舊) 6f | 0.677 | 0.688 |
  | event recall@50 | 21/75 (이진 24) | 24/80 (이진 23) |
  | 연도별 ROC (M_sa) | 22/23/24 = 0.69/0.62/0.78 | 0.69/0.63/0.82 |
- 부가 이득: ① P_H90 ≤ P_H150 정합 보장 — 기존 이진 쌍은 test 창의 47%
  (5,684/12,085)에서 위반 ② plain LR이라 확률 보정 확보(class-weight 인플레
  제거; 기존 "순위만 유효" 한계 해소) ③ 수술의 이중계상 제거.

## 단계별 판정

**P0-1 (2024 확장)** — test 사건 52→75 (H90) / 56→80 (H150). 게이트:
M-role이 2022-23 부분집합에서 anchor(0.643680/0.035973 등) 소수점 6자리
재현. 2024는 강한 해(M-role ROC 0.75/0.80) → 3년 종합 M-role 0.677/0.688.
**주의: 0.644→0.677은 모델 개선이 아니라 test 구성 변화**(더 정직·정밀한
추정치일 뿐).

**P0-2 (구조 교정)** — M_bf(pc_chronic 백필) **기각**: paired 음수
(−0.019/−0.004), rolling 방향 비일관(2022 −, 2023 +, 2024 −). 진단 프로브의
기대가 재검에서 죽음 — 정직 기록. **M_sa 채택**: paired dROC +0.014
[−0.007,+0.039] (H90) / **+0.017 [+0.001,+0.030] EXCL0** (H150) — 프로젝트
최초의 ROC-유의 additive 개선. rolling-origin **6/6 fold dROC 양(+)**
(H90 +0.003/+0.016/+0.032; H150 +0.014/+0.022/+0.028). ridge C=0.1 동일
(+0.0000) → 과적합 아님. 성분: prior_pc_rate 주도(+0.007/+0.008 단독),
H150 유의성은 +ncg_log 결합에서. 계수 해석: 현재 만성 사용량 낮음(−0.33) ×
직전 시즌 노출 높음(+0.10~0.15) = "감소 패턴"이 조기 시즌에도 관측 가능.

**P1 (미니 티어 3종, M_sa 위 additive)** — **전부 미채택**:
- prior_tjs (1f): dROC +0.005/+0.012, dPR +0.006/+0.006, 전부 CI 0 포함.
  계수 −0.34/−0.30 (보호 방향 — 사전 검증한 base rate 0.55×와 정합; 새
  인대/survivor 해석). H90 evrec@50 24→28 (+4)은 주목 — **사건 수 늘면
  1순위 재검정 후보**.
- pitch-mix (br_share_l15+dev): dROC **−0.030/−0.019 EXCL0 (유의 악화)**.
  Karnuta slider-usage 서사 전향 프레임 이전 실패 (Tier-2 블록 널과 정합).
- torque 가중 chronic: corr(tw, pc)=0.9998 → 완전 무효과 (±0.0002).
- rest 구조 (short_rest_30+gap_std_90): dPR **유의 음수** 양 H. 기각.

**P2 (모델 형태)** — hazard supermodel **채택** (조건: 유의 음수 아님;
실측 dROC +0.009 [−0.021,+0.036] / +0.004 [−0.006,+0.015] + 구조 이득).
role-stratified stitch **미채택** (+0.001/+0.015, CI 0 포함 — 확장 사건
수로도 미해소; 오류 분석 프로브의 재확인).

## 사후 적대 검증 (2026-07-10, 사용자 요청 "문헌 천장 초과 의심"; scripts/v_pblock.py)

- **V1 라벨 실재성 PASS**: 시트의 코호트-투수 수술 수 2022/23/24 = 30/36/29
  (연도 간 균질, 2024 스파이크 없음); 2024 사건 24건 전원 별개 투수·실제
  수술일, 중복 0.
- **V2 구성 PASS(부분)**: 2024 양성의 SP 비율 58% ≈ 2023(59%) — 역할 구성
  으로는 2024 강세(0.78-0.81)를 설명 못 함. 24 events 기준 연도 ROC 노이즈
  (±~0.09)가 상당 부분일 수 있음을 명시.
- **V3 M_sa 채택의 2024 의존성 없음**: paired dROC가 2022-23만(+0.010/
  +0.015)과 2024만(+0.027/+0.020)에서 모두 양 — 채택은 2024 유입과 무관.
- **V4 추정량 각주 (핵심)**: 0.70은 window-pooled ROC. 문헌(선수-시즌 AUC)
  비교용으로는 **6/1 단일 스냅샷 ROC ~0.69**, 날짜별 평균 0.67-0.69.
  연도별 스냅샷: 2022 0.64-0.67 / 2023 0.61-0.68 (**문헌 대역 0.61-0.67
  안**) / 2024 0.76-0.81 (강한 해 + 소사건 노이즈). **"문헌 천장을 넘었다"
  주장 금지** — 추정량이 다르고 비교 논문은 retrospective/CV. 정직 표현:
  "동일 대역 상단, 2024 포함 3년 평균 ~0.70".
- **V5 무결성 PASS**: 전 feature가 t 이전 경기만 사용(assert), ncg_log만
  2024에서 train 범위를 소폭 초과(5.88→6.33, 계수 −0.07로 영향 미미).
- **다중비교 반성**: P0-2는 2변형×2H=4 CI 중 1개 EXCL0 (familywise ~18%
  기대). M_sa 채택의 실질 근거는 단일 CI가 아니라 **방향 일관성 앙상블**
  (paired EXCL0 + rolling 6/6 + 연도 부분집합 전부 양 + ridge 불변) —
  R1의 M-role 채택과 동일한 기준임을 명시.
